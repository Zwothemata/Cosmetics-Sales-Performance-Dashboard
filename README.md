# Cosmetics-Sales-Performance-Dashboard
Interactive Google Sheets dashboard for analyzing retail sales across store locations. Includes filter-enabled tables and customizable bar charts to visualize total purchases by store. Provides clear insights into revenue distribution and top-performing outlets to support data-driven decisions.
